## UX Report — 2026-04-23

### Summary
Playwright UX tests are critically failing with an "invalid URL" error, preventing the application from loading and any subsequent UI/UX testing from occurring. This regression is highly correlated with recent infrastructure changes to the Playwright configuration and CI workflow. Additionally, a core `home.spec.js` test file has been removed, indicating a regression in test coverage.

### Findings

1.  **Critical Failure: Playwright Tests Cannot Load App**
    *   **Description:** The `app loads and shows splash screen` Playwright test fails at `page.goto('/')` with the error `Protocol error (Page.navigate): Cannot navigate to invalid URL`. This indicates that Playwright cannot resolve a valid base URL for the application, preventing any UX tests from running.
    *   **Likely Cause from Git Diff:** This is highly correlated with the recent changes to `playwright.config.js` (`2 +-`) and `.github/workflows/ux-test.yml` (`7 +++++-`). The commit subject "[infra] add Debug URL step before Playwright run" further suggests an ongoing issue or attempt to resolve a problem with the test environment's URL setup. The `playwright.config.js` file is responsible for defining the `baseURL` or `webServer` that Playwright uses to launch/connect to the app, while the workflow manages the environment and app startup.
    *   **Severity:** **High** - Blocks all automated UX testing, making it impossible to detect other UI/UX bugs or regressions.

2.  **Regression: Loss of Home Screen Test Coverage**
    *   **Description:** The `tests/ux/home.spec.js` file has been completely deleted.
    *   **Likely Cause from Git Diff:** Explicit deletion of `tests/ux/home.spec.js` (`12 -----------`). This indicates a direct removal of tests potentially covering the home screen's functionality or initial state.
    *   **Severity:** **Medium** - While not a direct app bug, it's a regression in QA and test coverage. If `app.spec.js` doesn't fully replace its functionality, critical parts of the home experience could now be untested.
