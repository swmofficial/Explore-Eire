## Context

CLAUDE.md is currently used as the single source of project memory for:
- Agents (both Architect and Implementer read it at session start)
- Gemini (gemini-analysis.js and gemini-audit.js slice it at 8000 chars)
- gemini-improve.js (the weekly product analyst run)

The problem: CLAUDE.md is 41,341 chars. The 8,000 char Gemini slice means
everything past line ~100 is invisible to the pipeline — the bug register,
build state, and roadmap are all beyond the slice. Meanwhile agents download
the full file and spend context budget on it.

The fix is a clean split into purpose-built files with different audiences.
