## Context

`tests/ux/sheets.spec.js` is supposed to validate the DataSheet bottom sheet (bug register
rule #22 — three-state spring gesture; rule #32 — two-layer architecture). Currently it:

1. Navigates to `/` (which loads the **dashboard** tab, NOT the map tab)
2. Waits 3 seconds
3. Takes a screenshot called `sheets-01.png` with **no assertion** about what's in it
4. Checks that zero console errors occurred

The DataSheet is a child of MapView and only renders on the map tab. The current spec never
navigates to the map tab, so it never exercises the DataSheet at all. The "no sheet errors"
test passes vacuously — it's testing the dashboard surface.

**What the code tells us:**
- `App.jsx` starts on `activeTab = 'dashboard'`. Map tab is hidden (visibility:hidden) but
  the MapView component IS mounted (always). DataSheet renders inside MapView.
- Clicking `aria-label="Map"` sets `activeTab = 'map'`, making MapView visible.
- In collapsed state the DataSheet shows a 60px peek at the bottom of the screen (the
  drag handle + context line). This is always visible after MapView becomes visible.
- DataSheet outer wrapper: `position: fixed; height: 100dvh; pointer-events: none; z-index: 20`
- DataSheet has no `aria-label` or `data-testid` on its container currently.

**Required change to DataSheet.jsx:**
Add `data-testid="datasheet"` to the outer wrapper div so tests can select it reliably.
This is a 1-line change with zero functional impact.

The outer wrapper div in `DataSheet.jsx` begins around line 257 (after the `return` in the
default export). It has inline style: `position: 'fixed', inset: 0, height: '100dvh',
pointerEvents: 'none', zIndex: 20`. Add `data-testid="datasheet"` as a prop on that div.

### 2. Rewrite sheets.spec.js

Replace the entire content of `tests/ux/sheets.spec.js` with:

```js
import { test, expect } from '@playwright/test';

test.beforeEach(async ({ page }) => {
  page._errors = [];
  page.on('console', m => { if (m.type() === 'error') page._errors.push(m.text()); });
  page.on('pageerror', e => page._errors.push(e.message));
  await page.goto('/');
  await page.waitForTimeout(2500);
  // Navigate to map tab — DataSheet only renders inside MapView
  await page.getByRole('button', { name: 'Map' }).click();
  await page.waitForTimeout(3000);
});

test('map canvas renders before DataSheet check', async ({ page }) => {
  const canvas = page.locator('canvas').first();
  await expect(canvas).toBeVisible();
  await page.screenshot({ path: 'tests/ux/screenshots/sheets-01.png' });
});

test('datasheet container is present in DOM', async ({ page }) => {
  const sheet = page.locator('[data-testid="datasheet"]');
  await expect(sheet).toBeAttached();
});

test('datasheet peek is within viewport at bottom', async ({ page }) => {
  const sheet = page.locator('[data-testid="datasheet"]');
  const box = await sheet.boundingBox();
  expect(box).not.toBeNull();
  // Outer wrapper covers full screen — bottom edge should equal viewport height
  const viewportHeight = page.viewportSize().height;
  expect(box.y + box.height).toBeGreaterThanOrEqual(viewportHeight - 10);
  await page.screenshot({ path: 'tests/ux/screenshots/sheets-02.png' });
});

test('no sheet errors on map tab', async ({ page }) => {
  expect(page._errors).toHaveLength(0);
});
```
